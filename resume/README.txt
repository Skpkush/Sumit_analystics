RESUME FOLDER
=============

Drop your resume PDF in THIS folder and name it EXACTLY:

    Sumit_Prajapat_Resume.pdf

The website's "Download resume" button and the command-palette
"download resume.pdf" action both link to:

    resume/Sumit_Prajapat_Resume.pdf

So as long as the file sits here with that exact name, the
download will work — no code changes needed.

If you want to use a different file name, update these two spots
in index.html:
  1. The hero <a> tag  (the "Download resume" button)
  2. The command palette resume action in the <script>

You can delete this README once your PDF is in place.
